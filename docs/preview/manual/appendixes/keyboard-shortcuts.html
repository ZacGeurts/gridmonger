Keyboard shortcuts¶

Keys on the numeric keypad are denoted with a «kp» prefix (e.g. kp 1, kp +, etc.)

Note

When it comes to keyboard shortcuts, Gridmonger uses the US keyboard layout, regardless of the keyboard layout and language settings of your operating system, or what the key labels on a non-US keyboard indicate. This is very similar to how most games handle the keyboard. For the more technically inclined, the program only cares about positional scancodes.

Movement keys¶

Standard movement keys
Arrow Keypad Vim

These keys are referred to as «Std move» from now on.

Combine with: Ctrl for 5-cell jumps; Shift to pan.

* NumLock must be off to use the keypad for movement.

← kp 4* H Left (West)
→ kp 6* L Right (East)
↑ kp 8* K Up (North)
↓ kp 2kp 5* J Down (South)
Diagonal movement keys
Keypad Vim

These keys are referred to as «Diag move» from now on.

* NumLock must be off to use the keypad for movement.
Combine with: Ctrl for 5-cell jumps; Shift to pan.

† With YUBN keys enabled. Combine with Shift to pan.

kp 7* Y† Up & left (Northwest)
kp 9* U† Up & right (Northeast)
kp 1* B† Down & left (Southwest)
kp 3* N† Down & right (Southeast)

Select (Mark) Mode¶

D+Move keys Draw selection
E+Move keys Erase from selection
R+Move keys Add rectangular area to selection
S+Move keys Subtract rectangular area from selection
A Select the whole level (mark all)
U* or X Clear selection
(unmark all)

* Except for YUBN keys

Y* or C Yank (copy) selection

* Except for YUBN keys

Ctrl+E Erase selection
Ctrl+F Fill selection
Ctrl+S Surround selection with walls
Ctrl+R Crop level to selection
Ctrl+M Move selection
Ctrl+C Set floor colour of selection
Esc or Ctrl+[ Cancel selection &
return to Edit Mode
